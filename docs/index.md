# Welcome to Rift

**Rift** is the ultimate infrastructure tool for your Discord server. It combines visual architecture design with high-fidelity music, global economy, and advanced moderation.

## Key Features

### :material-robot: Server Architect
Design your server visually using the **Wizard**. Create categories, channels, and roles in an interactive menu, then build them instantly.

### :material-music-note: Hi-Fi Audio
Stream high-quality music from YouTube, SoundCloud, and Bandcamp with advanced audio filters.

### :material-earth: Global Economy
A persistent economy system where your currency follows you across every server.

### :material-shield-check: Smart Moderation
Automated security, timed bans, logging, and a robust verification system.
