# Glossary

Definitions of terms used within Archanid's ecosystem.

## General Terms

### Cooldown
The waiting period required before a command can be used again.
*   *Example:* `/daily` has a 24-hour cooldown.

### Embed
A rich-media message box used by Discord bots to display information cleanly, often containing titles, descriptions, images, and color bars.

### Prefix
The character(s) used to trigger a bot command.
*   **Slash Command (`/`):** The modern, standard way to use Archanid.
*   **Text Prefix (`!`):** The legacy way (e.g., `!play`). Slash commands are recommended.

## Economy Terms

### Global Currency
A currency system where your balance is stored on the bot's central database, not the specific server's database. Your money travels with you.

### Net Worth
The total combined value of a user's Wallet + Bank + Inventory.

### RNG (Random Number Generator)
Used in gambling and loot drops. Determines the outcome of chance-based events.

## Music Terms

### Bridge
The technology Archanid uses to "play" Spotify links. Since Spotify does not allow direct bot streaming, the Bridge finds the equivalent audio on YouTube/SoundCloud.

### Scrobble
The act of recording a track you listened to and sending that data to Last.fm for statistics.

## Architecture Terms

### Blueprint
The temporary design configuration created in the `/wizard` before the server is actually built.

### Category
A folder in Discord that holds Text and Voice channels. Permissions set on a Category are usually inherited by the channels inside it.

### Template
A saved configuration of a server's structure (Channels, Roles, perms) that can be loaded later or shared.
