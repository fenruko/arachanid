# Command Reference

A list of Rift's primary command categories. Use `/help` in Discord for the most up-to-date syntax.

## Server Management
*   `/wizard` - Open the Server Architect interface to build channels and roles.
*   `/help` - View the help menu and command list.
*   `/invite` - Get the bot's invite link.

## Music
*   `/play` - Play music from YouTube, SoundCloud, or Bandcamp.
*   `/stop` - Stop playback and disconnect.
*   `/skip` - Skip the current song.
*   `/volume` - Adjust playback volume.
*   `/lyrics` - Fetch lyrics for the current song.

## Economy
*   `/work` - Earn currency by working.
*   `/daily` - Claim your daily reward.
*   `/balance` - Check your wallet and bank balance.
*   `/shop` - View or create server shops.
*   `/gamble` - Play games of chance (Slots, Blackjack).

## Moderation
*   `/ban` - Ban a user from the server.
*   `/kick` - Kick a user.
*   `/mute` - Timeout a user.
*   `/warn` - Issue a warning to a user.
*   `/purge` - Bulk delete messages.

## Social
*   `/profile` - View your custom profile card.
*   `/rank` - Check your level and XP.
*   `/leaderboard` - View the server or global leaderboard.
*   `/rep` - Give reputation points to another user.
