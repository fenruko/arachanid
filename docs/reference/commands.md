# Command Reference

A comprehensive list of all available commands.

!!! info "Slash Commands"
    All commands use the `/` prefix. Parameters marked with `?` are optional.

## General
| Command | Description | Permissions |
| :--- | :--- | :--- |
| `/help` | Shows the help menu. | None |
| `/ping` | Checks bot latency. | None |
| `/invite` | Get the bot invite link. | None |
| `/support` | Get the support server link. | None |

## Architect
| Command | Description | Permissions |
| :--- | :--- | :--- |
| `/wizard` | Opens the Server Builder GUI. | Admin |
| `/architect backup` | Backs up server layout. | Admin |
| `/architect load` | Loads a server layout. | Admin |
| `/architect nuke` | Deletes all channels/roles. | Owner |

## Economy
| Command | Description | Permissions |
| :--- | :--- | :--- |
| `/balance` | Check wallet/bank. | None |
| `/work` | Earn active income. | None |
| `/daily` | Claim daily reward. | None |
| `/deposit` | Move cash to bank. | None |
| `/withdraw` | Move cash to wallet. | None |
| `/shop view` | View server shop. | None |
| `/shop create` | Create a shop item. | Admin |
| `/pay` | Transfer money to user. | None |

## Music
| Command | Description | Permissions |
| :--- | :--- | :--- |
| `/play` | Queue a song/playlist. | None |
| `/skip` | Vote to skip. | None |
| `/stop` | Clear queue and disconnect. | DJ / Admin |
| `/volume` | Set volume (0-100). | DJ / Admin |
| `/filter` | Set audio filter. | DJ / Admin |
| `/lyrics` | Fetch lyrics for current song. | None |

## Moderation
| Command | Description | Permissions |
| :--- | :--- | :--- |
| `/ban` | Ban a user. | Ban Members |
| `/kick` | Kick a user. | Kick Members |
| `/mute` | Timeout a user. | Moderate Members |
| `/purge` | Bulk delete messages. | Manage Messages |
| `/lock` | Lock channel. | Manage Channels |
| `/slowmode` | Set rate limit. | Manage Channels |
| `/warn` | Warn a user. | Moderate Members |
| `/history` | View user's mod log. | Moderate Members |
